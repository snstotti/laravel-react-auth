
## Installation

Clone the repo locally:

```sh
git clone https://github.com/snstotti/laravel-react-auth.git
cd laravel-react-auth
```

Install PHP dependencies:

```sh
composer install
```

Install NPM dependencies:

```sh
cd ./client
npm install
```

Build assets:

```sh
npm run start
```

Setup configuration:

```sh
cp .env.example .env
```

Run database migrations:

```sh
php artisan migrate
```

Run artisan server:

```sh
php artisan serve
```
